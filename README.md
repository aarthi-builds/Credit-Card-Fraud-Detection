# Credit Card Fraud Detection System

A secure, web-based Fraud Detection System built using Python (Flask) and SQLite. 

## Key Features
- **Fraud Monitoring:** Detects invalid access attempts.
- **Secure Authentication:** Validates user identity through multi-field verification.
- **Real-time Feedback:** Provides instant visual alerts.

## Tech Stack
- **Backend:** Python, Flask
- **Frontend:** HTML, CSS, JavaScript
- **Database:** SQLite

## How to Setup & Run

### Prerequisites
- Python installed.
- Flask library installed.

### Steps to Run
1. **Clone the repository:**
   `git clone https://github.com/aarthi-builds/Credit-Card-Fraud-Detection.git`

2. **Navigate to the project folder:**
   `cd Credit-Card-Fraud-Detection`

3. **Install dependencies:**
   `pip install flask`

4. **Run the application:**
   `python app.py`

5. **Access the system:** Open your browser and navigate to `http://127.0.0.1:5000`

## Database Setup
- Ensure the fraud_detection.db file is placed in the root folder.
